GCLIB_DLL_EXPORTED GReturn GCALL GLost ( GCon  g)

Checks for a lost connection.

Attention
GLost() checks if a device has been lost. Devices are considered "lost" when the suddenly disapear which can happen when a controller is powered off or the ethernet cord is disconnected. GLost should be called periodically and the results handled accordingly.
Parameters
gConnection's handle.
Returns
The success status or error code of the function.
